# Weather Summarizer

This is just a simple repo for a weather summarizer django backend app using celery, rabbitmq, redis and flower.

## Pre-requisite

- `docker-compose`
- `make`

## Getting started

```bash
# Use make to run docker-compose
make recompose
```

Access Flower web UI at <http://localhost:5555>
